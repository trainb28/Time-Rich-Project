PRESS LOGOS
===========
Drop five image files in this folder and the site upgrades automatically.
If a file is missing, that outlet falls back to a clean text wordmark, so
the page never breaks. You can add them one at a time.

Required filenames:
  fortune.png
  forbes.png
  hbr.png
  fastcompany.png
  inc.png

Specs:
  - PNG with transparent background (or SVG, just change the extension in index.html)
  - Roughly 60px tall at 2x, so about 120px tall source
  - Any color. The site applies a grayscale filter so they sit together evenly.

Where to get them:
  Most publications have a press or media kit page with approved logo files.
  Search "[publication] press kit" or "[publication] brand assets".
  Using publication logos in an "as featured in" bar is standard practice
  when you were genuinely featured, which you were.
